# Survival-Shooter
Survival Shooter Game

## For Clone
  ```
  git clone --recursive https://github.com/Vinaysangwan/Survival-Shooter.git
  ```

## Build:
  ```
  cmake -S . -B build
  ```

## Compile:
  ```
  cmake --build build
  ```

## Using Makefile
  ```
  make build
  ```
  ```
  make run
  ```
